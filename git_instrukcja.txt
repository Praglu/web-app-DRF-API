Instrukcja do gita:

- tworzymy nowy folder
- w środku git bash
$ git init --- tworzy repozytorium
$ git status --- pokazuje status naszego repozytorium
$ git add
$ git commit -m"Komentarz( dobrze, żeby było wiadomo o co chodzi w commicie)
_________________________________________
$ git config --global user.name "Imię"   \
$ git config --global user.email "email" /
________________________________________/

$ git log --- historia commitów
$ git add -A --- wszystkie pliki dodane do stageing area
$ git commit -am --- dodanie wszystkich do stageingu i skomentowanie
$ git branch --- pokazuje na jakim branchu jesteśmy i jakie są dostępne
$ git checkout -b"nazwa brancha"
$ git checkout master --- powrót na mastera
$ git merge <nazwa brancha> --- dodanie commitów na mastera

Po stworzeniu repozytorium na githubie

$ git remote add origin <link do repo>
( $ git branch -M main --- jeśli chcemy z mastera przerzucić wszystko na maina)
$ git push -u origin master

Jeśli chcemy pobrać z githuba z innego repo

$ git clone <link do repo>
